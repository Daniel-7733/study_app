# Learning Plan

## Phase 1: Basic Window

Goal: Make the app open.

- Create the main Tkinter window.
- Add title and size.
- Add navigation buttons.

## Phase 2: Timer Page

Goal: Build the focus timer.

- Input total study time.
- Input break interval.
- Start, pause, reset.
- Show break reminder.

## Phase 3: Notes Page

Goal: Save what was learned.

- Add title input.
- Add note text box.
- Save note.
- Show saved notes later.

## Phase 4: SQLite

Goal: Store notes permanently.

- Create database connection.
- Create notes table.
- Insert notes.
- Read notes.

## Phase 5: Flashcards

Goal: Create question and answer cards.

- Add question input.
- Add answer input.
- Save flashcard.
- Link flashcard to a note later if desired.

## Phase 6: Review Page

Goal: Show cards that need review.

- Add next_review_date.
- Show due cards.
- Update card after review.

## Phase 7: Polish

Goal: Make the app pleasant to use.

- Better layout.
- Better colors.
- Edit and delete buttons.
- Session history.
